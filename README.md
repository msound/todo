# todo
This is a simple todo app written in Go and HTMX.

# Development
Use [air](https://github.com/cosmtrek/air) for hot reloading the application.

Install `air` using:

```
go install github.com/comstrek/air
```

Make sure `air` is in your $PATH.

Then, simply run:

```
air
```
